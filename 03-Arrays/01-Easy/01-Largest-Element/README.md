# Largest Element in Array

## Problem Statement
Given an array of integers, find the largest element.

## Example
Input: [2, 5, 1, 3, 0]  
Output: 5

## Optimal Approach
1. Assume first element is largest.
2. Traverse the array.
3. If current element is greater than largest, update largest.
4. Return largest.

## Time Complexity
O(n)

## Space Complexity
O(1)

## Key Learning
- Basic array traversal
- Maximum value tracking
