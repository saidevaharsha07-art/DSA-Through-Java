# 05 - Basic Maths

## What is Basic Maths in DSA?
Basic Maths means solving problems using numbers, digits, divisors, factors, and simple mathematical logic.

This section builds the foundation for:
- Number theory
- Hashing
- Recursion
- Arrays
- GCD/LCM based problems
- Prime number problems

## Topics Covered

### 01. Count Digits
Learn how to extract and count digits from a number.

### 02. Reverse Number
Learn how to reverse digits using modulo and division.

### 03. Palindrome Number
Check whether a number is same after reversing.

### 04. GCD
Find the greatest number that divides two numbers exactly.

### 05. Armstrong Number
Check whether a number is equal to the sum of powers of its digits.

### 06. Print Divisors
Find all numbers that divide a given number exactly.

### 07. Prime Number
Check whether a number has only two factors: 1 and itself.
