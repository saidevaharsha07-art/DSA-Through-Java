# Level 1 - Check Palindrome Number

## Question

Write a Java program to check whether a given number is a palindrome or not.

## Example

Input:
121

Output:
Palindrome

Input:
123

Output:
Not Palindrome

## Logic

Reverse the number and compare it with the original number.

If both are equal, it is a palindrome.
