# Level 1 - Check Prime Number

## Question

Write a Java program to check whether a given number is Prime or Not Prime.

## Example

Input:
13

Output:
Prime Number

Input:
12

Output:
Not Prime Number

## Logic

A prime number has exactly two factors:

1 and itself

Count the factors of the number.

If factor count is 2, it is prime.

## Hint

Use:

```java
if(n % i == 0)
```

Count how many divisors the number has.
