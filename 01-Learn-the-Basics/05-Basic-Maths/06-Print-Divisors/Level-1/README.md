# Level 1 - Print All Divisors

## Question

Write a Java program to print all divisors of a given number.

## Example

Input:
12

Output:
1 2 3 4 6 12

## Logic

A number is a divisor if it divides the given number completely.

Check every number from 1 to n.

## Hint

Use:

```java
if(n % i == 0)
```

Print i whenever the condition is true.
