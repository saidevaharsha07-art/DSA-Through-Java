# 03 Two Sum

## Logic

Find two elements whose sum is equal to the target value.

## Steps

1. Take array input.
2. Take target value input.
3. Select one element.
4. Compare it with all remaining elements.
5. Check whether their sum equals the target.
6. If found, print the pair or indices.
7. Stop searching.

## Formula

arr[i] + arr[j] = target
