# 19 Check if Array is Sorted

## Logic

Check whether every element is smaller than or equal to the next element.

## Steps

1. Take array input.
2. Traverse the array from index 0 to n - 2.
3. Compare current element with next element.
4. If current element is greater than next element, array is not sorted.
5. If no such case is found, array is sorted.
6. Print the result.
