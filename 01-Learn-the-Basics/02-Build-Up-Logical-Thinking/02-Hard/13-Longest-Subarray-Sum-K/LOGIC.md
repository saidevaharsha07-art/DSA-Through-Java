# 13 Longest Subarray Sum K

## Logic

Find the length of the longest subarray whose sum is equal to K.

## Steps

1. Take array input.
2. Take value K as input.
3. Calculate the sum of subarrays.
4. Check whether the sum equals K.
5. Track the maximum length found.
6. Update the answer whenever a longer subarray is found.
7. Print the maximum length.

##Formula

Subarray Sum = K
