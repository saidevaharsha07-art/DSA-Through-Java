# 02 Sum of Digits

## Logic

Extract each digit of the number and add them together.

## Steps

1. Take input n.
2. Create sum = 0.
3. Find the last digit using n % 10.
4. Add the digit to sum.
5. Remove the last digit using n / 10.
6. Repeat until n becomes 0.
7. Print the sum.

## Example

Input: 1234

1 + 2 + 3 + 4 = 10
