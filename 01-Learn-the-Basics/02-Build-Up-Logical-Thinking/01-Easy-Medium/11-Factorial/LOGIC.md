# 11 Factorial

## Logic

Factorial means multiplying all natural numbers from 1 to n.

## Steps

1. Take input n.
2. Create fact = 1.
3. Run a loop from 1 to n.
4. Multiply each number with fact.
5. Print fact.

## Formula

n! = 1 × 2 × 3 × ... × n

## Example

Input: 5

5! = 1 × 2 × 3 × 4 × 5 = 120

Output: 120
