# 05 Palindrome Number

## Logic

A number is a palindrome if it remains the same after reversing.

## Steps

1. Take input n.
2. Store the original number.
3. Reverse the number.
4. Compare original number and reversed number.
5. If both are equal, it is a palindrome.
6. Otherwise, it is not a palindrome.

## Example

Input: 121

Reverse: 121

Output: Palindrome Number
