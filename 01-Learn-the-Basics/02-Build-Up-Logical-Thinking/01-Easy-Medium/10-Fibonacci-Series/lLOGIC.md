# 10 Fibonacci Series

## Logic

Each number in the Fibonacci series is the sum of the previous two numbers.

## Steps

1. Take input n.
2. Initialize:
   - a = 0
   - b = 1
3. Print a and b.
4. Find next = a + b.
5. Update:
   - a = b
   - b = next
6. Repeat until n terms are printed.

## Example

Input: 7

Series:
0 1 1 2 3 5 8

Output:
0 1 1 2 3 5 8
