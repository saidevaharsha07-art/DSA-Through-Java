# 20 Pattern

## Pattern

```text
*        *
**      **
***    ***
****  ****
**********
****  ****
***    ***
**      **
*        *
```

## Logic

Print a butterfly pattern using stars and spaces.

## Steps

1. Take input n.
2. Print increasing stars on both sides.
3. Print decreasing spaces in the middle.
4. Continue until the center row.
5. Reverse the process for the lower half.
6. Print the complete butterfly pattern.
