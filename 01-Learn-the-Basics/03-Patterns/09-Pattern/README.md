# 09 Pattern

## Pattern

```text
    *
   ***
  *****
 *******
*********
*********
 *******
  *****
   ***
    *
```

## Logic

Combine a pyramid and an inverted pyramid to form a diamond.

## Steps

1. Print the upper pyramid.
2. Print the lower inverted pyramid.
3. Maintain proper spacing.
4. Use odd number of stars.
5. Print the complete diamond pattern.
