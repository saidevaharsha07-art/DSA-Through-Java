# 06 Pattern

## Pattern

```text
1 2 3 4 5
1 2 3 4
1 2 3
1 2
1
```

## Logic

Print numbers in decreasing order for each row.

## Steps

1. Take input n.
2. Run outer loop from n to 1.
3. Run inner loop from 1 to current row value.
4. Print the inner loop value.
5. Move to next line after each row.
