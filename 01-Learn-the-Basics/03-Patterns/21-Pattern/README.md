# 21 Pattern

## Pattern

```text
*****
*   *
*   *
*   *
*****
```

## Logic

Print stars on the border and spaces inside.

## Steps

1. Take input n.
2. Print stars in the first and last rows.
3. Print stars in the first and last columns.
4. Print spaces in the inner positions.
5. Continue for all rows.
6. Print the hollow rectangle pattern.
