# 14 Pattern

## Pattern

```text
A
A B
A B C
A B C D
A B C D E
```

## Logic

Print alphabets from A up to the current row.

## Steps

1. Take input n.
2. Run outer loop for rows.
3. Run inner loop from A to current row character.
4. Print alphabets in sequence.
5. Move to the next line after each row.
