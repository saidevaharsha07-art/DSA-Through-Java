# 22 Pattern

## Pattern

```text
4444444
4333334
4322234
4321234
4322234
4333334
4444444
```

## Logic

Print concentric rectangular layers of numbers.

## Steps

1. Take input n.
2. Determine the minimum distance of each position from the border.
3. Use that distance to calculate the value to print.
4. Print the same value for positions in the same layer.
5. Continue until all layers are completed.
6. Print the rectangular number pattern.
