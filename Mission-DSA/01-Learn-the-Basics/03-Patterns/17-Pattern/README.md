# 17 Pattern

## Pattern

```text
    A
   ABA
  ABCBA
 ABCDCBA
ABCDEDCBA
```

## Logic

Print alphabets in increasing order and then decreasing order to form a pyramid.

## Steps

1. Take input n.
2. Print leading spaces.
3. Print increasing alphabets from A.
4. Print decreasing alphabets back to A.
5. Move to the next line after each row.
