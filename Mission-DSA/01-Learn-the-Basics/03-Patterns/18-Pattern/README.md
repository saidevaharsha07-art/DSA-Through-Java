# 18 Pattern

## Pattern

```text
E
E D
E D C
E D C B
E D C B A
```

## Logic

Print alphabets in reverse order starting from the last alphabet.

## Steps

1. Take input n.
2. Start printing from the last alphabet.
3. Print one extra alphabet in each row.
4. Continue in descending order.
5. Move to the next line after each row.
