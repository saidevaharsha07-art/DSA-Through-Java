# 10 Pattern

## Pattern

```text
*
* *
* * *
* * * *
* * * * *
* * * *
* * *
* *
*
```

## Logic

Print an increasing triangle followed by a decreasing triangle.

## Steps

1. Take input n.
2. Print increasing star triangle.
3. Print decreasing star triangle.
4. Use nested loops for both parts.
5. Print the complete half diamond pattern.
