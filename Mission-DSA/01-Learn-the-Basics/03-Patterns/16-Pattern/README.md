# 16 Pattern

## Pattern

```text
A
B B
C C C
D D D D
E E E E E
```

## Logic

Print the same alphabet repeatedly in each row.

## Steps

1. Take input n.
2. Start with alphabet A.
3. Print the current alphabet row number times.
4. Move to the next alphabet for the next row.
5. Continue until n rows are printed.
