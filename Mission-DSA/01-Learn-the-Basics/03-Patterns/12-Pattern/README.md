# 12 Pattern

## Pattern

```text
1        1
12      21
123    321
1234  4321
1234554321
```

## Logic

Print increasing numbers on the left and decreasing numbers on the right with spaces in between.

## Steps

1. Take input n.
2. Print increasing numbers from 1 to row number.
3. Print spaces in the middle.
4. Print decreasing numbers back to 1.
5. Move to the next line after each row.
