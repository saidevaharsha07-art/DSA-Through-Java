# 19 Pattern

## Pattern

```text
**********
****  ****
***    ***
**      **
*        *
*        *
**      **
***    ***
****  ****
**********
```

## Logic

Print a symmetric hollow pattern using stars and spaces.

## Steps

1. Take input n.
2. Print decreasing stars on both sides.
3. Print increasing spaces in the middle.
4. After reaching the center, reverse the process.
5. Print the complete symmetric pattern.
