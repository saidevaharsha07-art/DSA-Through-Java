# 11 Pattern

## Pattern

```text
1
0 1
1 0 1
0 1 0 1
1 0 1 0 1
```

## Logic

Print alternating 0s and 1s based on row and column positions.

## Steps

1. Take input n.
2. Determine the starting value for each row.
3. Alternate between 0 and 1.
4. Print values for the current row.
5. Move to the next line after each row.
