# 01 Pattern

## Pattern

```text
* * * * *
* * * * *
* * * * *
* * * * *
* * * * *
```

## Logic

Print a square pattern where each row contains n stars.

## Steps

1. Take input n.
2. Run outer loop from 1 to n for rows.
3. Run inner loop from 1 to n for columns.
4. Print star in every column.
5. Move to next line after each row.
