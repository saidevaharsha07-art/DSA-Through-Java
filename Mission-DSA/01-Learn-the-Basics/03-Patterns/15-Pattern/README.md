# 15 Pattern

## Pattern

```text
A B C D E
A B C D
A B C
A B
A
```

## Logic

Print alphabets in decreasing order.

## Steps

1. Take input n.
2. Run outer loop from n to 1.
3. Print alphabets from A up to current row length.
4. Decrease the number of characters in each row.
5. Move to the next line after each row.
