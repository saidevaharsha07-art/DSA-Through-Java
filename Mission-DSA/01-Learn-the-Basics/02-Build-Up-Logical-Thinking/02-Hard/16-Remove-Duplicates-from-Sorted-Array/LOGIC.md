# 16 Remove Duplicates from Sorted Array

## Logic

Remove repeated elements from a sorted array and keep only unique elements.

## Steps

1. Take sorted array input.
2. Create a position variable for unique elements.
3. Traverse the array from the second element.
4. Compare current element with previous element.
5. If both are different, store current element at the unique position.
6. Increase the unique position.
7. Print the unique elements.
