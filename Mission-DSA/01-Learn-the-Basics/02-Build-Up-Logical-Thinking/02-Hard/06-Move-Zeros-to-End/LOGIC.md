# 06 Move Zeros to End

## Logic

Move all zero elements to the end of the array while maintaining the order of non-zero elements.

## Steps

1. Take array input.
2. Create a position variable for non-zero elements.
3. Traverse the array.
4. Whenever a non-zero element is found, place it at the current position.
5. Increment the position.
6. After processing all elements, fill the remaining positions with zeros.
7. Print the updated array.
