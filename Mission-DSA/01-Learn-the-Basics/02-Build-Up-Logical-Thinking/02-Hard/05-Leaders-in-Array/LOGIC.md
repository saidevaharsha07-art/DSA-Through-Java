# 05 Leaders in Array

## Logic

Find all elements that are greater than every element present on their right side.

## Steps

1. Take array input.
2. Select one element.
3. Compare it with all elements on its right.
4. If no greater element exists on the right, it is a leader.
5. Repeat for all elements.
6. Print all leader elements.

## Example

Input: 16 17 4 3 5 2

Output: 17 5 2
