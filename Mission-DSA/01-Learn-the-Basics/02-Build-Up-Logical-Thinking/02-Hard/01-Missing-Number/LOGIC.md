# 01 Missing Number

## Logic

Find the missing number from a sequence containing numbers from 1 to n.

## Steps

1. Take array input.
2. Calculate the expected sum from 1 to n.
3. Calculate the actual sum of the array.
4. Subtract actual sum from expected sum.
5. The result is the missing number.
6. Print the missing number.

## Formula

Expected Sum = n × (n + 1) / 2

Missing Number = Expected Sum − Actual Sum
