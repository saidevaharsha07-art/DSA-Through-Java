# 12 Power of a Number

## Logic

Find how many times the base is multiplied by itself.

## Steps

1. Take input base and power.
2. Create result = 1.
3. Run a loop from 1 to power.
4. Multiply result with base each time.
5. Print result.

## Formula

base^power

## Example

Input: base = 2, power = 3

2³ = 2 × 2 × 2 = 8

Output: 8
