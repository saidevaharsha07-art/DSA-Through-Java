# 07 Prime Number

## Logic

A prime number has exactly two factors: 1 and itself.

## Steps

1. Take input n.
2. If n ≤ 1, it is not prime.
3. Check divisibility from 2 to √n.
4. If any number divides n completely, it is not prime.
5. Otherwise, it is prime.

## Example

Input: 13

Factors: 1, 13

Output: Prime Number
